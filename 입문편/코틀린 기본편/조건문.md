# 조건문

- if else는 똑같음

- when 문도 사용가능

```
fun main() {
    val score = 20;
    when (score) {
        100 -> {
            println("100");
        }
        90 -> {
            println("90");
        }
        80 -> {
            println("80");
        }
        else -> {
            println("no");
        }
    }
}
```
