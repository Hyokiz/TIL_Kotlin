# 레이아웃(Layout)이란?

- 뷰 그룹의 일종으로, 뷰나 다른 레이아웃을 배치시키는 역할

## 5가지 레이아웃

1. 리니어 레이아웃

- 선형 레이아웃. 
- 수직 혹은 수평으로 차례로 적용

2. 상대적 레이아웃

- 부모 레이아웃이나, 다른 레이아웃을 기준으로 상대적으로 배치

3. 컨스트레인트 레이아웃

- 뷰들의 수직, 수평 방향에 제약을 준다.

4. 테이블 레이아웃

- 앨범 같이 정렬하고자 할때 사용

5. 프레임 레이아웃

- 뷰들을 액자처럼 쌓아놓음.
- 여러 뷰들을 추가하더라도 가장 나중에 추가한 것을 가장 위에 배치
